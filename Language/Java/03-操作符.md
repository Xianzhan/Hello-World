# 优先级

Operators|Precedence
:---|:---
postfix|`expr++` `expr--`
unary|`++expr` `--expr` `+expr` `-expr` `~` `!`
multiplicative|`*` `/` `%`
additive|`+` `-`
shift|`<<` `>>` `>>>`
relational|`<` `>` `<=` `>=` `instanceof`
equality|`==` `!=`
bitwise AND|`&`
bitwise exclusive OR|`^`
bitwise inclusive OR|`|`
logical AND|`&&`
logical OR|`||`
ternary|`expr?expr:expr` 
assignment|`=` `+=` `-=` `*=` `/=` `%=` `&=` `^=` `|=` `<<=` `>>=` `>>>=`
