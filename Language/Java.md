## 反射

反射是 Java 语言中一个相当重要的特性，它允许正在运行的 Java 程序观测，甚至是修改程序的动态行为。

反射在 Java 中的应用十分广泛。开发人员日常接触到的 IDE 便运用了这一功能，当然，大部分使用语法树来实现。
