# 代码

```c
#include <stdio.h>   // 提供 printf()
#include <stdlib.h>  // 提供 system()

// C 标准, 没有参数时使用 void 关键字
int main(void) 
{
    printf("hello world");

    system("pause");
    // return 不写时, 编译器会自动添加
    return 0;
}
```
